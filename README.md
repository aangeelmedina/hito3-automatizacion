## 📋 Descripción General

Este repositorio contiene dos proyectos de automatización inteligente desarrollados con n8n, Ollama y PostgreSQL:

- **Proyecto A (Alejandro):** Sistema RAG que procesa documentos, los vectoriza con Qdrant y permite hacer preguntas sobre su contenido.
- **Proyecto B (Ángel):** Chatbot conversacional que analiza el mensaje del usuario y decide automáticamente qué herramienta externa consultar.
